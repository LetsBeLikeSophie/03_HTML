HSL Valuse

hsl(hue, saturation, lightness)
CSS3에서 RGB색상에 대한 hue, saturation, lightness 색상값을 추가
hue : 색조 : 0 ~ 360 : 0 또는 360은 red, 120은 green, 240은 blue
saturatin : 채도 : 0% ~ 100% : 0%는 회색빛, 100%는 원래의 완전한 색상
lightness : 명도 : 0% ~ 100% : 0%는 검정색, 100%는 흰색
background-color:hsl(120,100%, 50%) : green

HSLA, RGBA

HSL, RGB에 ahpha(투명도)가 추가 됨
hsla(hue, saturation, lightness, alpha)
rgba(red, green, blue, alpha)
